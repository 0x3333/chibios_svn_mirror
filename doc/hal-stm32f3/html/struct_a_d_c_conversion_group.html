Detailed Description

Conversion group configuration structure.

This implementation-dependent structure describes a conversion operation.

Note
The use of this configuration structure requires knowledge of STM32 ADC cell registers interface, please refer to the STM32 reference manual for details.

◆ cfgr

uint32_t ADCConversionGroup::cfgr

ADC CFGR register initialization data.

Note
The bits DMAEN and DMACFG are enforced internally to the driver, keep them to zero.
The bits ADC_CFGR_CONT or ADC_CFGR_DISCEN must be specified in continuous mode or if the buffer depth is greater than one.
